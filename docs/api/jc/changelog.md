# 版本更新记录

## 2.2.0
`2018-12-09`

* 新增导航栏菜单切换，默认选中第一个子菜单
* 新增一级菜单可以点击页面事件
* 优化如果无二级菜单，左边菜单栏自动隐藏
* 优化表单只验证格式，不需要必填的bug
* 优化文件可以批量上传
* 新增树自定义点击事件
* 新增窗口收缩功能
* 新增按钮提交前回调
* 新增静态表格弹出窗口编辑

`2018-10-06`
